# Global Layoffs Analysis Dashboard

## Project Overview
This project analyzes global layoffs data using SQL for data cleaning and transformation and Power BI for visualization and business insights.

## Tools Used
- MySQL
- Power BI
- SQL
- Excel/CSV

## Key Analysis
- Monthly layoffs trends
- Top companies by layoffs
- Industry-wise layoffs
- Country analysis
- Company stage analysis

## Key Insights
- Layoffs peaked during late 2022 and early 2023
- Technology companies experienced major workforce reductions
- Post-IPO companies showed the highest layoffs
